# WGA SDK Kotlin

**Version:** 1.0-SNAPSHOT

## Description

This is the WGA SDK Kotlin library.

## Installation

You can use the WGA SDK by adding the following dependency to your `build.gradle.kts` file:

```kotlin
dependencies {
    implementation("kr.neoventures.wga:wga-sdk-kotlin:1.0-SNAPSHOT")
}
```

## Building from Source

To build the project from source, you need the following:
- JDK 21
- Gradle

Clone the repository and run the following command:

```bash
./gradlew build
```
